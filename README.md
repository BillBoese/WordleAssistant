"# Wordle Assistant" 
A command line python utility to assist with solving the wordle.

Enter your guess and the results and the assistant returns a list of possible answers.

It can be run interactively to help solve the Wordle, or it can be run against all 5 letter words
to determine the result profile for that word, i.e. average number of guesses to solution 
and distribution of results.
